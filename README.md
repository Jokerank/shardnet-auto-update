# shardnet-auto-update
### Near shardnet node auto update with script

```
If you want all scripts with auto and manual commit update do this:
git clone https://github.com/Jokerank/shardnet-auto-update.git

If you need auto
wget https://github.com/Jokerank/shardnet-auto-update/blob/main/update.sh

If you need manual (it's just asking you commit to use, that's all)
wget https://github.com/Jokerank/shardnet-auto-update/blob/main/manualcommitupdate.sh

Just run it like usual script.
Examples:

sh update.sh
OR
sh manualcommitupdate.sh
